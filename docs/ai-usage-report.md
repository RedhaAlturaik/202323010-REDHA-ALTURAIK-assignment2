## AI Enhancement

### AI Tools Used

#### ChatGPT
ChatGPT was used as a support tool during the development of this assignment.  
It acted as a learning assistant to help clarify concepts, suggest improvements, and review parts of the code.

How it was used:
- to discuss possible ways to make the portfolio more interactive
- to review JavaScript logic for filtering projects dynamically
- to help improve the dark mode toggle using `localStorage`
- to provide suggestions for structuring documentation files

AI suggestions were not copied directly. The code was reviewed, tested, and adjusted to fit the structure of my own project.

### How AI Supported Development

AI was used mainly for:
- explaining JavaScript logic when implementing dynamic features
- reviewing code structure and suggesting improvements
- helping organize documentation such as the README and technical documentation

The final implementation was written, modified, and tested manually to ensure it worked correctly with the existing portfolio code.

### Understanding of AI Output

Before using any AI suggestions, I made sure to understand how the code worked.

For example, I reviewed and tested:
- how project filtering works using JavaScript arrays and `.filter()`

AI was used as a learning aid and development partner, while the final implementation reflects my own understanding and adjustments.
