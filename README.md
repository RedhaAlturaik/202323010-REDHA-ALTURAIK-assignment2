# REDHA AZIZ – Portfolio Website (Assignment 2)

## Project Description

This is my personal portfolio website built using HTML, CSS, and JavaScript.  
It is based on Assignment 1 and improved in Assignment 2 by adding more interactivity, dynamic content, better user feedback, and smoother user experience.

Main features include:

- Responsive navigation menu
- Dark mode with saved user preference using localStorage
- Dynamic project cards generated with JavaScript
- Live project search/filter
- Active navigation highlight while scrolling
- Contact form validation with instant feedback
- Smooth transitions and hover effects

## Setup Instructions (Run Locally)

1. Download or clone the project folder.
2. Make sure the file structure is correct:
   - `index.html`
   - `css/styles.css`
   - `js/script.js`
3. Open `index.html` in any modern browser.

No additional installation or dependencies are required.
